# ISDiscordBot


## Card Format
Cards should be in a json format like so:

### Objects of Power
```
{
    "title": "TITLE",
    "colour": "COLOR",
    "level": "LEVEL",
    "effect_depletion": "EFFECT DEPLETION",
    "object_depletion": "OBJECT DEPLETION",
    "form": "FORM",
    "description": "DESCRIPTION"
}
```
