# ISDiscordBot


## Card Format
Cards should be in a json file in a `decks` directory in the same. 

### General Spells
```
{
    "title": "TITLE",
    "colour": "COLOR",
    "level": "LEVEL",
    "depletion": "DEPLETION",
    "description": "DESCRIPTION"
}
```

### Incantations
```
{
    "title": "TITLE",
    "colour": "COLOR",
    "level": "LEVEL",
    "depletion": "DEPLETION",
    "description": "DESCRIPTION"
}
```

### Ephemera
```
{
    "title": "TITLE",
    "colour": "COLOR",
    "level": "LEVEL",
    "depletion": "DEPLETION",
    "form": "FORM",
    "description": "DESCRIPTION"
}
```

### Kindled Items
```
{
    "title": "TITLE",
    "colour": "COLOR",
    "level": "LEVEL",
    "form": "FORM",
    "description": "DESCRIPTION"
}
```

### Objects of Power
```
{
    "title": "TITLE",
    "colour": "COLOR",
    "level": "LEVEL",
    "effect_depletion": "EFFECT DEPLETION",
    "object_depletion": "OBJECT DEPLETION",

```
